## 📝 CUSTOMIZATION TIPS

### Change Colors:
- Edit `tailwind.config.js`
- Cyan: `#00D9FF`
- Pink: `#FF006E`

### Update Content:
- Projects: Edit `components/Projects.jsx`
- Skills: Edit `components/Skills.jsx`
- Profile info: Edit `components/Profile.jsx`

### Add More Sections:
1. Create new component in `/components`
2. Import in `pages/index.jsx`
3. Add to nav in `components/Navigation.jsx`
